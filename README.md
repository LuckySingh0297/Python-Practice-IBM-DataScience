# 🐍 Python Practice – IBM Data Science Professional Certificate

This repository contains my hands-on practice work completed as part of the **IBM Data Science Professional Certificate**.

The goal of this repository is to build a strong foundation in Python by solving practical problems, working with real datasets, and understanding core programming concepts used in data science.

---

## 📌 Topics Covered

### 1. Python Basics

* Variables and Data Types (int, float, string, boolean)
* Type casting and basic operations
* Input and output handling

### 2. Operators

* Arithmetic Operators
* Comparison Operators
* Logical Operators
* Assignment Operators

### 3. Data Structures

* Lists
* Tuples
* Sets
* Dictionaries

### 4. Control Flow

* If, Else, Elif statements
* Nested conditions
* Loops (for, while)

### 5. Functions

* Function definition and calling
* Parameters and return values
* Built-in functions

### 6. File Handling

* Reading and writing files using `with open()`
* Working with text files
* Handling unstructured data

### 7. Working with Libraries

* Introduction to NumPy
* Introduction to Pandas
* Basic data manipulation

### 8. Data Handling

* Loading datasets from files and APIs
* Cleaning and transforming data
* Converting unstructured data into structured format

### 9. Web Data Extraction

* Using APIs with `requests`
* Basic web scraping concepts


## 🎯 Objective

* Strengthen Python fundamentals
* Build problem-solving skills
* Prepare for data science and analytics roles
* Practice real-world data handling scenarios
